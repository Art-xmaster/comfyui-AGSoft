## Installation

To install comfyui-AGSoft Custom Nodes for ComfyUI, you can follow the following steps:

1. Go to ComfyUI/custom_nodes dir in terminal (cmd)
2. git clone https://github.com/Art-xmaster/comfyui-AGSoft.git
3. Restart ComfyUI
