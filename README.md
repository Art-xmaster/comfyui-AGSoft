## Установка

Через ComfyUI Manager:
1. Откройте ComfyUI Manager
2. Найдите "comfyui-AGSoft" в списке пользовательских узлов
3. Установите

Или вручную:

cd ComfyUI/custom_nodes
git clone https://github.com/Art-xmaster/comfyui-AGSoft.git
